# Adjustable Bar Clamp (F-Clamp) Assembly 

A complete 3D CAD assembly model of an adjustable bar clamp. This repository contains the master assembly file along with all individual component files, suitable for mechanical reference, 3D printing, or digital rendering.

## Tool Specifications

* **Clamp Type:** F-Clamp / Adjustable Bar Clamp
* **Mechanism:** Threaded screw pressure with a sliding locking jaw

## Bill of Materials (Components)

This bottom-up assembly is composed of 4 main modeled parts:
1. **Fixed Jaw & Bar (Blue):** The main structural rail.
2. **Sliding Jaw (Yellow):** Adjusts along the bar to fit different workpiece sizes.
3. **Threaded Handle (Red):** The screw mechanism to apply clamping force.
4. **Swivel Pad (Grey):** Articulating pad to distribute pressure evenly.

## Software & Tools Used

* **Modeling Software:** [AutoCAD]

## Repository Contents

* `/src`: Contains the native CAD assembly file and all individual part files.
* `/exports`: Contains universal 3D formats for sharing:
  * `.STEP` (Recommended for importing the full assembly into other CAD software)
  * `.STL` (Individual parts exported for 3D printing)
* `/images`: Contains visual references, including `Screenshot 2026-06-26 170746_2.png`.

## How to Use This Model

1. Clone the repository: `git clone https://github.com/yourusername/your-repo-name.git`
2. If opening the native files in `/src`, ensure all part files remain in the same folder as the main assembly file to prevent broken links.
3. To view in different software, import the `.STEP` file located in the `/exports` folder.
